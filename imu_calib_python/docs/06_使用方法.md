# 06 使用方法

## 1. 环境要求

- Python 3.11+（当前开发环境已在 3.14 下验证）
- `numpy`
- `scipy`
- `pandas`
- `matplotlib`
- `pytest`

安装依赖：

```bash
pip install -r requirements.txt
```

## 2. 运行完整演示

```bash
python examples/main_demo.py
```

或：

```bash
python -m imu_calib demo
```

## 3. 从 CSV / manifest 运行

```bash
python examples/run_from_csv.py data/example_dataset_bundle
```

只运行 `static.csv`：

```bash
python examples/run_from_csv.py data/example_dataset_bundle/static.csv
```

## 4. 单模块运行示例

### 4.1 加速度计标定

```python
from imu_calib.io.load_csv_data import load_csv_data
from imu_calib.tasks.run_acc_calibration import run_acc_calibration

data = load_csv_data("data/example_dataset_bundle/acc_poses.csv")
task = run_acc_calibration(data)
```

### 4.2 温度模型：仅陀螺

```python
from imu_calib.io.load_csv_data import load_csv_data
from imu_calib.tasks.run_temperature_fit import run_temperature_fit

data = load_csv_data("data/example_dataset_bundle/static.csv")
task = run_temperature_fit(data, target="gyro")
```

### 4.3 温度模型：陀螺 + 加速度

```python
from imu_calib.io.load_csv_data import load_csv_data
from imu_calib.tasks.run_acc_calibration import run_acc_calibration
from imu_calib.tasks.run_temperature_fit import run_temperature_fit

data = load_csv_data("data/example_dataset_bundle/static.csv")
acc_task = run_acc_calibration(data)
temp_task = run_temperature_fit(
    data,
    target="both",
    Ca=acc_task["result"]["Ca"],
    ba=acc_task["result"]["ba"],
)
```

说明：

- 若 `static.csv` 中没有足够多的姿态切换静止段，`ba(T)` 可能返回 `invalid` 或 `low_confidence`

## 5. 运行期补偿

```python
from imu_calib.runtime.apply_imu_calibration import apply_imu_calibration

corrected = apply_imu_calibration(raw_data, calib_param)
```

当前关键输出：

- `corrected["acc"]`
- `corrected["gyro"]`
- `corrected["baT"]`
- `corrected["bgT"]`

## 6. 如何判断运行成功

可先看：

- `task["success"]`
- `task["message"]`
- `task["warnings"]`

完整流程再看：

- `results.validation["summary"]`
- `print_calibration_summary(results)` 输出

## 7. 常见错误

### `ba(T)` 无法拟合

常见原因：

- 没有提供固定 `Ca`
- 温度跨度太小
- 静止段提取失败
- 每个温度点的多姿态覆盖不足

### 只有 `static.csv` 但加速度温度模型失败

这通常不是代码错误，而是数据本身不具备可观测性。请检查：

- 是否真的有多个静止姿态
- 每个姿态持续时间是否足够
- 温度 bin 内是否覆盖多个方向
