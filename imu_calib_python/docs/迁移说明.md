# 迁移说明

## 1. 本次迁移内容

温度模块已从“只支持 `bg(T)`”扩展为：

- `bg(T)`
- `ba(T)`

同时明确当前阶段的边界：

- 固定 `Ca`
- 不拟合 `Ca(T)`

## 2. 旧实现

旧实现只支持：

- 陀螺零偏温漂 `bg(T)`

运行期只有：

- `get_gyro_bias_from_temperature(...)`

## 3. 新实现

新增：

- `fit_accel_temperature_bias_model(...)`
- `fit_temperature_bias_models(...)`
- `get_accel_bias_from_temperature(...)`

任务层：

- `run_temperature_fit(...)` 现在支持：
  - `gyro-only`
  - `accel-only`
  - `both`

## 4. 输出结构变化

温度结果现在统一包含：

- `reference_temperature`
- `temperature_range`
- `extrapolation_mode`
- `gyro`
- `acc`

其中 `gyro` / `acc` 内部至少包含：

- `enabled`
- `valid`
- `low_confidence`
- `model_type`
- `coeff`
- `metrics`
- `message`

## 5. 运行期变化

加速度运行期从固定 bias 模式扩展为：

$$
a_{corr} = C_a(a_{raw} - b_a(T))
$$

如果没有有效 `ba_model`，则自动退化为固定 `ba`。

## 6. 当前不包含的内容

以下不在本次实现范围内：

- `Ca(T)`
- `Ka(T)`
- `Ma(T)`
- 陀螺 `Cg(T)`

若后续要继续扩展，应从当前 `fit_temperature_bias_model.py` 的两层结构继续发展，而不是把 bias、scale、misalignment 混在一个函数里。
