# 04 补偿算法与运行流程

## 1. 统一运行期入口

当前统一补偿入口：

- `imu_calib/runtime/apply_imu_calibration.py`

输入：

- `raw_data["gyro"]`
- `raw_data["acc"]`
- 可选 `raw_data["temp"]`

核心参数：

- `bg`
- `Ca`
- `ba`
- `Cg`
- 可选 `Gg`
- 可选 `temp.bg_model`
- 可选 `temp.ba_model`

## 2. 完整补偿链路

```mermaid
flowchart TD
    A[原始 gyro / acc / temp] --> B[根据 temp 求 ba(T)]
    B --> C[加速度去 bias: a_raw - ba(T)]
    C --> D[加速度矩阵补偿: a_corr = Ca * (a_raw - ba(T))]
    A --> E[根据 temp 求 bg(T)]
    E --> F[陀螺去 bias: omega_m - bg(T)]
    D --> G[f_term = a_corr]
    G --> H[Gg * f_term]
    F --> I[去除 g-敏感度项]
    H --> I
    I --> J[solve(Cg, ...)]
    J --> K[gyro_corrected]
    D --> L[acc_corrected]
```

## 3. 加速度补偿

当前实际执行：

$$
a_{corr} = C_a(a_{raw} - b_a(T))
$$

代码对应：

```python
baT, _ = get_accel_bias_from_temperature(temp, ba_const, ba_model)
acc_bias_removed = acc - baT
acc_calibrated = (Ca @ acc_bias_removed.T).T
```

说明：

- 若没有有效 `ba_model`，则退化为常温固定 `ba`
- `Ca` 固定，不随温度变化

## 4. 陀螺补偿

当前正向模型：

$$
\omega_m = C_g\omega_{ref} + b_g(T) + G_g f_{term} + n_g
$$

运行期逆补偿：

$$
\hat{\omega}_{ref} = \mathrm{solve}\left(C_g,\ \omega_m - b_g(T) - G_g f_{term}\right)
$$

代码对应：

```python
bgT, _ = get_gyro_bias_from_temperature(temp, bg_const, bg_model)
gyro_bias_removed = gyro - bgT
g_term = (Gg @ f_term.T).T
gyro_model_removed = gyro_bias_removed - g_term
gyro_calibrated = np.linalg.solve(Cg, gyro_model_removed.T).T
```

## 5. 温度补偿顺序

当前顺序是：

1. 根据 `temp` 求 `ba(T)`
2. 执行 `a_corr = Ca * (a_raw - ba(T))`
3. 根据 `temp` 求 `bg(T)`
4. 执行陀螺去 bias
5. 若 `Gg` 有效，扣除 `Gg * f_term`
6. 执行 `solve(Cg, ...)`

说明：

- 当前 `f_term` 使用已补偿后的 `a_corr`
- 这是当前 Python 主线的真实实现

## 6. 温区外策略

运行期统一支持：

- `clamp`
- `extrapolate`
- `warn_and_clamp`

默认：

- `warn_and_clamp`

模型文件中保存：

- `reference_temperature`
- `temperature_range`
- `extrapolation_mode`

运行期会根据这些字段对 `bg(T)` / `ba(T)` 做一致求值。

## 7. 调试中间量

`apply_imu_calibration(...)` 当前会返回：

- `bgT`
- `baT`
- `bias_removed_gyro`
- `gsens_removed_gyro`
- `f_term`
- `g_term`

这些字段可直接用于验证补偿链路是否符合预期。

## 8. 当前边界

- 当前只实现 `bg(T)` 与 `ba(T)`
- 当前不实现 `Ca(T)` / `Cg(T)` / `Gg(T)`
- 若没有温度列，运行期会自动退化到常温固定 bias 模式
